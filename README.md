# Finance Data Analysis (Excel)

## Overview
This project involves analysing financial data using Excel to identify spending patterns, trends, and key insights.

## Features
- Analysed income and expense data across multiple categories  
- Structured and cleaned data for effective analysis  
- Identified patterns in spending behaviour over time  
- Created visualisations to illustrate financial trends  

## Tools Used
- Microsoft Excel  

## Key Insights
- Analysed 45+ financial transactions  
- Identified major spending categories such as food, bills, and transport  
- Evaluated monthly spending trends and behavioural patterns  

## What I Learned
- Data cleaning and structuring  
- Applying Excel formulas for analysis  
- Interpreting and visualising financial data
